My Portfolio

Portfolio

This collection of analyses were made using the statistical programming language, R. The purpose of each project is to analyze one data set and create a predictive machine learning model from it. Each project will follow a common structure. It will start with an initial exploratory data analysis (EDA). The EDA will be filled with data visualizations and inital Q&As to help understand a data set. The EDA is then followed by a model exploration. Different machine learning alogrithms will be used to create predictive models. Models will then be compared against each other and the most accurate predictive model will be determined. The conclusion section will provide an executive summary for each project. Each project will have its own repository on my github.
